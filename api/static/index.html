Getting started
1Go to Get a key and enter the invite code you were given. Save the key somewhere safe — it can't be recovered.
2Go to Manage files, drag in your documents (PDF, Word, text, images), and click Index files. Takes a minute depending on size.
3Come back to Search and ask anything in plain English. Your key is stored in this browser — paste it again on any new device.
⚠ Indexing is currently unavailable — the embedding service is offline. Search still works for already-indexed documents. Try again later or contact the server host.
